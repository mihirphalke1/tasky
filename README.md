## üöÄ Quick Start

### Prerequisites

- Node.js 18+
- Firebase account
- Modern web browser

### Installation

```bash
# Clone the repository
git clone https://github.com/mihirphalke1/tasky.git
cd tasky

# Install dependencies
npm install

# Configure Firebase
cp .env.example .env

# Start development server
npm run dev
```
