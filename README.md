## ğŸ® How to Use

### **Creating Tasks**

- **Quick Add**: Press `Cmd/Ctrl + J` anywhere to add a task
- **Smart Input**: Type naturally - "Call mom tomorrow at 2pm high priority"
- **Traditional Mode**: Use structured forms for detailed task creation

### **Focus Sessions**

1. **Enter Focus Mode**: Press `Cmd/Ctrl + Shift + Enter`
2. **Set Your Intention**: Define what you want to accomplish
3. **Stay Focused**: Use Focus Lock to prevent distractions
4. **Track Progress**: Monitor completed tasks and time spent

### **Pomodoro Technique**

1. **Start Timer**: Press `P` to open Pomodoro panel
2. **Begin Session**: Press `Space` to start/pause timer
3. **Stay Informed**: View shortcuts without interrupting your flow
4. **Take Breaks**: Automatic break reminders between sessions
